Welcome to the Inventory System

Select an option below to continue.


View Inventory
Displays a list of all available supplies and equipment. Users can browse items, check availability, and view basic details for each item.
Checkout Items
Allows users to select an available item and check it out for use. Once completed, the item is marked as unavailable until it is returned.
Return Items
Provides a way for users to return items they have previously checked out. Returning an item updates its availability in the system.
Inventory Status
Shows a summary of current inventory levels. This section helps staff quickly see what items are available, checked out, or running low.
Transactions
Displays a history of all checkout and return activity for tracking and auditing purposes. (Future development)
Add Items
Allows administrators to add new inventory items into the system. (Future Development)
Checked Out Items
Displays all items currently checked out across the system. (Future Development)
